Meu primeiro site com CSS

Bem-vinda ao meu primeiro projeto de desenvolvimento web. Este site foi criado como parte do meu processo de aprendizagem, com o objetivo de explorar e aplicar os conceitos fundamentais de HTML e CSS.

Ao longo desta página, estou a testar diferentes estilos, cores e formas de organização de conteúdo. Cada elemento foi pensado para me ajudar a compreender melhor como funciona a construção visual de um site.

Este espaço representa o início da minha jornada na programação e no desenvolvimento web. Com dedicação e prática, pretendo evoluir cada vez mais, criando projetos mais completos, funcionais e visualmente apelativos.

Este conteúdo está em destaque para demonstrar a aplicação de estilos específicos, mostrando como é possível personalizar partes individuais de uma página
